# 크리퍼 감지기 (Creeper_Detector)
게임 마인크래프트(Minecraft)의 화면에서 크리퍼(Creeper)를 인식하고,
크리퍼 주변에 예상 폭발 피해 범위(블록, 엔티티)를 표시해주는 프로그램입니다.

# 필요 환경
## Python
- Python 3.12.0+
- OpenCV 4.9.0+
- numpy 1.26.4+
- pyautogui 0.9.54+
- pygetwindow 0.0.9+

## Minecraft
- [Minecraft Java Edition 1.20.6](https://www.minecraft.net/)
